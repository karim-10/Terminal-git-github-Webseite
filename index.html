Was ist Terminal?

Ein Terminal ist textbasiert und dient
als Befehlszeilenschnittstelle (CLI),
in die Ihr Eure Befehle eingeben kö
Eine Shell nimmt diese Befehle entgegen
und weist das Betriebssystem an,sie
auszuführen.

pwd

Print Working Directory

ls

list

ls -1

list in einer Spalte

ls -l

List - long format

ls -a

List mit versteckten Dateien

Was ist Git?

Git ist ein verteiltes Versionskontrollsystem zur Verwaltung von Quellcode.Ok, was ist also Versionskontrolle? Einfach ausgedrückt, ist die Versionskontrolle ein System zur Verfolgung von Änderungen an Dateien. Wenn Sie Dateien ändern, zeichnet das Versionskontrollsystem jede Änderung auf und speichert sie. Auf diese Weise können Sie jederzeit eine frühere Version Ihres Codes wiederherstellen.

git init

git add .

git commit
